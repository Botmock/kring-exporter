# Kring Exporter

This script extends the `@botmock/export` `BaseExporter` in order to generate `.lg` files from Botmock projects.

## How to use the generated `.lg` file In Bot Framework

See their [docs](https://docs.microsoft.com/en-us/azure/bot-service/bot-builder-concept-language-generation?view=azure-bot-service-4.0&tabs=javascript)
