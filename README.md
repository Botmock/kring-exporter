# Kring Exporter

This script extends the `@botmock/export` `BaseExporter` in order to generate `.lg` files from Botmock projects.

## The `.lg` file format

The `.lg` files generated by this script contain content for [Language Generation](https://docs.microsoft.com/en-us/azure/bot-service/bot-builder-concept-language-generation?view=azure-bot-service-4.0&tabs=javascript), Microsoft has released in order to allow developers to

- build adaptive cards
- maintain unity of expression and voice in their bot
- have context-based expression execution
- define variations on a phrase
